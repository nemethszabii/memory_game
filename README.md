# 🧠 React Memory Game

A sleek, responsive card-matching memory game built with **React JS** and styled using **Tailwind CSS**. Test your memory by finding all matching pairs of cards in the fewest moves possible!

---

## 🚀 Features

*   **Move Counter & Timer:** Track your performance and challenge yourself to beat your high score.
*   **State Management:** Built purely with React hooks (`useState`, `useEffect`, `useRef`).
*   **Instant Reset:** Quickly restart the game with shuffled cards at any time.

---

## 🛠️ Tech Stack

*   **Frontend Library:** [React JS](https://react.dev/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Build Tool:** Vite
